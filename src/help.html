Getting Started

1. Add RSS Feeds

Add one or more RSS feeds on the "Feeds" page. For example: 

example.com/feed

These RSS feeds will be scanned periodically for updates.

2. Add Link Types

Any link that is not in the list of "Link Types" will be ignored. Therefore you should make sure that the type of links you would like to add is in there. For example: Assuming the RSS feed gives you links like: 

http://example.com/example_movie_download.rar

This means you would need to add:

example.com

to the list of link types.

3. Add Filters

Add "Filters" that match the title of the RSS entries. Lets say the title is: 

Great Example Movie that has 720p resolution and DTS audio and other useless info

Then an appropriate filter could look like this (regex enabled):

Example Movie.*(720|1080).*(DTS|AC3)

See this tutorial for more details on how regular expressions work in perl.

If you often use very similar filters, you can set up template filters in the "Filter configuration" on the "Filters" page.
